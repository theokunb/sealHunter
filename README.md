# Пародия игры Sealhunter с использование 2D

## Управление
- WASD - хотьба по сторонам
- F - стрельба
- B - купить оружие
- G - поменять оружие

## Правила игры

- Нужно не пропускать врагов налево
- Игрок может иметь макимум 2 оружия
